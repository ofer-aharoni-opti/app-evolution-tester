# App Evolution Tracker v1.0

**Comprehensive evolution tracker with smart size management**

## Features
- ✅ Contextual PR analysis with accumulated knowledge
- ✅ Architectural pattern recognition and learning
- ✅ Technology evolution tracking over time  
- ✅ Smart data size management (files stay <8KB)
- ✅ Evolution insights and predictions
- ✅ Team convention learning
- ✅ Automatic content rotation and archiving

## Data Location
`.app-knowledge/evolution/` (in your repository)

## Installation
1. Copy `SKILL.md` to `.claude/skills/app-evolution-tracker/SKILL.md`
2. Create `.app-knowledge/evolution/` in your repository
3. Use: `claude --skill app-evolution-tracker "analyze PR 123"`

## Intelligence Growth
- **PRs 1-10:** Learning basic patterns
- **PRs 11-25:** Recognizing team conventions
- **PRs 26-50:** Understanding evolution phases  
- **PRs 50+:** Predicting evolution needs and strategic insights

## File Size Management
- `architecture-overview.md`: Max 5KB
- `recent-timeline.md`: Max 8KB (last 20 changes)
- `architectural-patterns.md`: Max 6KB
- `current-insights.md`: Max 5KB
- Total folder: ~25KB (stays constant over time)

**Comprehensive intelligence without repository bloat!**
